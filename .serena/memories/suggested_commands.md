# Suggested Commands
- Preview: `python3 -m http.server 8080`, then open `http://localhost:8080`.
- Search/check content: `rg -n '<pattern>' index.html README.md AGENTS.md`.
- Deploy via Vercel from the repository only with `.vercelignore` intact; for manual upload, create a clean temporary directory containing only `index.html` and approved WebP assets.
